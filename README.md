> Taqtile Danger-js Plugin

## Usage

Install:

```sh
yarn add danger-plugin-taqtile-dangerjs-plugin --dev
```

At a glance:

```js
// dangerfile.js
import taqtileDangerjsPlugin from 'danger-plugin-taqtile-dangerjs-plugin'

taqtileDangerjsPlugin()
```
