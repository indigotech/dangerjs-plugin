> Taqtile Danger-js Plugin

## Usage

Install:

```sh
yarn add taqtile-dangerjs-plugin --dev
```

At a glance:

```js
// dangerfile.js
import taqtileDangerjsPlugin from 'taqtile-dangerjs-plugin'

taqtileDangerjsPlugin()
```
